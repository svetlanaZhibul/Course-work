Что это за приложение? 
Данное приложение представляет собой консольный обработчик изображений.
С его помощью можно изменять файлы формата(?) jpg, png, gif путем:
    применения эффектов ("черно-белый", "спокойный", 
"ретро", "насыщенный", "сетчатый", "пастель" и т.д.),
    изменения тона (яркость, контрастность),
    корректирования (изменение размера).

Работа с приложением. 
При запуске приложения появляется консоль.
Для изменения изображения вводится набор команд. Например, 
'код' "apply retro 1.jpg 2.jpg" осуществляет применение эффекта 
"ретро" к файлу 1.jpg с сохранением изменений в 2.jpg.
Результат обработки, файл 2.jpg, можно найти в папке "converted" (по умолчанию) 
для выходных изображений, находящейся в одной директории с приложением.
