Регистрация и авторизация:
	1. Программа доступна только для зарегистрированных пользователей. (Для подтверждения регистрации в папке с программой должен присутстововать файл register.txt c регистрационным ключом.?)
	2. При отсутсвии ключа при использовании программа выводит сообщение об ошибке
	3. Получение регистрационного ключа:
		a. При вызове программы с параметром --register пользователь переходит к диалогу сбора регистрационной информации
		b. В диалоге пользователь вводит свой email
		c. Прогамма отправляет данные пользователя на сервер авторизации
		d. На указанный email пользователю приходит письмо с регистрационным ключом
		е. Пользователь вводит ключ, тем самым подтверждая и завершая авторизацию
		
Положения о работе с программой:
	1. На входе в программу пользователь передает абсолютный или относительный путь к дирерктории с графическими файлами для обработки (в ходе работы путь может быть неоднократно изменен)
	2. На выходе программа создает в указанной дирректории папку "results", где хранятся результаты обработки данных 
	3. В запросе для обработки файлов используются сочетания команд для соответствующих изменений
	4. Один и тот же запрос может быть применен к нескольким файлам одновременно
	5. Команды для обработки можно передать следующими способами:
		a. Как параметры вызова программы
		b. В произвольном файле txt формата
	
Дополнительно:
	1. Предусмотрено сохранение истории запросов за последние 24 часа
	2. Существует опция просмотра возможных команд и их описаний
